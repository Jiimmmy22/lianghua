功能特点

📊
全球市场支持

📈
缠论分析

集成缠论分析功能，提供专业的技术分析工具

💾

支持数据本地缓存，提高查询效率

安装说明

# 克隆仓库
git clone https://github.com/Jiimmmy22/lianghua.git
cd lianghua

# 创建虚拟环境
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# 或
.venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt

使用方法

支持的股票代码格式

  • A股代码（6位数）：如 601318
  • 港股代码（5位数或更少）
  • 美股代码（如AAPL）
  • ETF基金：在数字前添加"e"，如e510050

启动应用

# 启动应用
python app.py

# 访问网址
http://localhost:8087
